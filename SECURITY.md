# Security Policy

## Supported Versions

We only provide security updates for the latest version of the project.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it by opening a private issue or contacting the maintainers directly.

Please do not report security vulnerabilities through public GitHub issues.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
