# Startup Workflow

This workflow will start the backend and frontend servers for the Minor Project.

1. Ensure MongoDB is running on your system.
2. Start the Backend Server:
   - Navigate to `server/`
   - Run `npm run dev`
3. Start the Frontend Application:
   - Navigate to `client/`
   - Run `npm run dev`

> [!TIP]
> Use the `/start` command in the future to keep this project running.
